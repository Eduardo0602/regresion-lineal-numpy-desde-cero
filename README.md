## Descripción

Implementación de regresión lineal sin librerías de ML — solo NumPy y matemáticas.
Aplicado a datos sísmicos reales de Ecuador (USGS, 2010–2025) para modelar la
geometría de la zona de subducción: predecir la profundidad de un sismo a partir
de su ubicación geográfica.

## Dataset

- **Fuente:** [USGS Earthquake Catalog](https://earthquake.usgs.gov/fdsnws/event/1/)
- **Período:** 2010–2025
- **Región:** Ecuador (lat -5.5° a 2.5°, lon -82° a -75°)
- **Registros:** 1,188 sismos con magnitud ≥ 2.0
- **Variable objetivo:** Profundidad del hipocentro (km)

## Tecnologías

Python 3.11 · NumPy · Pandas · Matplotlib · Seaborn · Scikit-learn (solo verificación)
